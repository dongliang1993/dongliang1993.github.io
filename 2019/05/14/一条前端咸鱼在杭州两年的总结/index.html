May 14, 2019

其实很久之前就想写一个总结了，只是自己太懒，脑子里想想都觉得很累的事情，付诸笔端的动力就更没有了。

今天看到群里一个老哥写了一篇到上海半年的总结帖，自己也突然想写点什么。

从 16 年到现在，工作也大概两年时间了，从一个啥也不懂的门外汉，到现在还算熟练的搬砖工，回顾自己这两年来，也觉得

我之前大部分的时间都在开发中后台的管理项目，所以移动端的开发经验比较欠缺。近三个月自己承担了很多移动端的业务，从中学到了很多，比如一元抢购改版是基于移动端的活动，所以遇到很多浏览器兼容方面的坑，比如不同手机和平台下，滚动事件有差别，所以写滚动定位和切换 tab 的时候遇到有些手机不能切换的问题，还有复制到剪切板，对于低版本的 ios 系统是不支持的以及在 ios 上视频播放会自动全屏，这些问题都是自己在之前的开发经历中没有遇到的
开发了一系列的活动运营页面，比如《一元抢购小姐姐》、《求生女神》、《年度盛典》、《网吧活动》、《冒泡排行榜》等
研究了前端 ssr 框架（nuxt.js、next.js），为后面做 ssr 做技术方案探索
搭建了基于 hybrid app 的前端项目架构 Fish，在之前的基础上增加了多线程打包压缩、公共代码提取、异步加载等功能，减少了打包体积和打包时间 。在新架构的整理的时候收获比较大，之前 web 端在移动端这块有两套代码， maopao-share 和 maopao-active。前者对应 app 内嵌 h5 以及分享到 app 外的需求，后者是基于 jquery + artTemplate 进行纯活动页的开发。随着 4.0.0 改版以及目前活动需求的增多，之前的架构有点不太适应之后的要求。所以这段时间，我们设计了一个偏底层的基础架构，然后在此之上，分出来了三个项目架构，一个用于和 app 端进行 hybrid 混合开发，一个用来开发活动相关，另一个是用来开发较复杂的运营活动以及承担分享的功能
冒泡 4.0.0 内发现页面使用了 h5 进行 hybrid 混合开发，因为发现页面这边是沉浸式的，所以和 app 端商量 webview 改用全屏，这样定制性更强一些， 后面和 app 端进一步商量了 jsBridge 交互协议。

2019 年工作计划

继续深入学习 React 、Vue 以及生态圈相关的库/插件，尝试使用 Angular 。语言基础当然重要，但是这种提升效率的框架/库也是开发中必不可少的环节。一方面大家都在说要重基础，将框架束之高阁，另一方面多少人用框架写出的屎一样的代码。想要在工作中更好更快的开发就离不开这些，一旦涉及到效率、优化，就免不得对底层有所了解。所以明年还是要不断的熟悉这些东西，去看一些源码。和前端框架有关的源码大概看过：Vuex、Redux、Zant（部分）、Axios、Underscore、Lodash（部分）、Anamate.css 等。个人目标是明年看完 React Router 、 React 核心，Vue Router、Vue 核心
加强 js 语言基础积累，拓展计算机基础。经过这段时间的工作和面试、被面试，有个发现：前端是人和人差别巨大的一个职业。有的人做编译器、编辑器，有的人写框架、写库，有的人复制粘贴都会弄错段落。前端从之前的刀耕火种到现在的跌跌撞撞，肯定有很多混乱的东西，这个靠一两个人解决不了，只能交给时间。我们能做到的就是扎实基础，韩涛大大之前说过一句话让我记忆犹新：广积粮、高筑墙、缓称王，对应在程序员方面就是语言基础扎实、底层知识过关，提高行业壁垒，这样才能一点点的好起来。前端届很有意思，行业壁垒完全靠层出不穷的新东西，跟不上的被淘汰，几年后再淘汰自己。所以，我们需要做的就是，扎实语言基础（多看书多写代码，比如红宝书、《你不知道的 js》等），补上算法和计算机网络
继续学习 Node.js，拓展相应后端只是。经过一段时间的考虑，我发现自己对纯前端（搞样式、小程序、宣传页、活动页）没有太大兴趣，所以后面会逐渐把更多的时间和精力放到偏后端方面，因此 Node.js 是一个很好的切入点（Golang 也挺不错），可以先从这个点入手，系统的学一下后端的东西，然后再考虑其他语言。毕竟语言只是个工具而已。目前自己只是大概用过 exporess、koa，看过 koa2 的源码。后面打算是系统的学一下相关知识，比如数据库、多线程等
提升自己的软实力。之前我一直有个错觉：只有专注技术提升才是正道。最近看了一些书渐渐明白了，技术只是一种技能，是实现目标的一种手段，仅此而已。现在社会的工作很少能靠单一技能支撑了，程序员也不例外，所以技术之外也有很多东西要去学习。比如，提搞自己的沟通能力和团队协作能力，跳出开发思维，从团队和产品的角度去考虑问题；多看一些其他种类的书；坚持学习英语；学会写博客总结，学会营销自己等
锻炼身体。从毕业到现在，我胖了 50 斤。。。有一次加班到凌晨 3 点多，做出租车回家的时候明显感觉心跳加速，赶紧设置了紧急联系人，生怕猝死。所以还是狗命要紧，哪怕不去跑步、举铁，花个一小时走走路也是好的 😂
